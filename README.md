###DIGITAL MIRROR
###

A lightweight OSINT tool designed 
to track down usernames across
various social media platforms
and web services.

## Features
- **Breoad Coverage:** Checks major platforms
and niche websites (the ones where people feel
"too safe").

- **Anti-Bot Measures:** Includes custom User-Agent 
headers to mimic real browser behavior.

- **Fast & Reliable:** Optimized timeouts to
prevent the script from handing on
unresponsive servers.

- **Minimalist:** Clean CLI output for better 
readability.
